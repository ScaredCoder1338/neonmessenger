# Telegram Web Prototype

Реализация Telegram-like веб-приложения с интерфейсом и реальной отправкой сообщений.

## Функциональность

- ✅ Современный интерфейс, похожий на Telegram Web
- ✅ Список чатов с последними сообщениями
- ✅ Область переписки с сообщениями
- ✅ Отправка и получение сообщений в реальном времени
- ✅ WebSocket сервер для обмена сообщениями
- ✅ Уведомления о новых сообщениях
- ✅ Индикатор онлайн-статуса
- ✅ Создание новых чатов
- ✅ Адаптивный дизайн
